## Why this matters for SOC
A SOC analyst must understand IT fundamentals in order to:
- Recognize abnormal system behavior
- Identify where logs are generated
- Understand how systems communicate within a network
- Analyze alerts with proper technical context

## SOC Example
If a system generates repeated authentication failures, an analyst must understand:
- Whether the system is a client or a server
- Which operating system is involved
- Which log sources should be investigated
